## 页面结构

每份知识清单按章节分为若干小节，统一使用以下排版规范：

**重要程度标签（每个小节标题右侧）：**

| 标签     | 含义                 |
| -------- | -------------------- |
| `必考` | 历年必出，需完全掌握 |
| `重点` | 核心概念，高频考点   |
| `常考` | 较常出现，需理解     |
| `了解` | 补充背景，了解即可   |

**文本标注：**

- 橙色高亮 = 核心术语（key point）
- 蓝色高亮 = 推理与解释（explanation）
- 引用块（灰色左边框）= 易错点提示

**公式渲染：**页面通过 MathJax 渲染 LaTeX 公式，初次打开需联网加载，之后可离线使用。

## 使用方式

1. 下载 `.html` 文件到本地
2. 用浏览器（推荐 Chrome / Edge）直接打开
3. 页面左侧/顶部有**目录（TOC）**，可快速跳转到各小节
4. 复习时重点关注标注了 `必考` 和 `重点` 的小节，以及引用块中的易错点
